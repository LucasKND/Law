# Template Institucional para Advocacia

![Logo do Projeto](../assets/balance.png)

## 📝 Descrição

Template institucional moderno e elegante desenvolvido especificamente para escritórios de advocacia e profissionais da área jurídica. O site apresenta um design sofisticado com elementos visuais que remetem ao universo jurídico, como a balança da justiça, combinados com uma interface intuitiva e responsiva.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estruturação semântica da página
- **CSS3**: Estilização completa e animações
- **JavaScript**: Interatividade e funcionalidades dinâmicas
- **Boxicons**: Biblioteca de ícones
- **Google Fonts**: Fonte Poppins para tipografia moderna

## ✨ Funcionalidades

- **Design Responsivo**: Layout adaptável para dispositivos móveis, tablets e desktops
- **Elementos Animados**: Animações sutis para melhorar a experiência do usuário
- **Carrossel de Áreas de Atuação**: Apresentação visual das diferentes áreas do direito atendidas
- **Seção de FAQ**: Perguntas frequentes com sistema de acordeão
- **Estatísticas Visuais**: Cartões informativos com métricas de sucesso
- **Mapa Interativo**: Localização do escritório integrada com Google Maps
- **Footer Informativo**: Com dados de contato, redes sociais e horário de funcionamento

## 📋 Seções do Site

1. **Header**: Navegação principal com logo e botões de contato/menu
2. **Hero**: Seção de destaque com slogan principal e chamada para ação
3. **Sobre**: Apresentação do escritório e seus valores
4. **Áreas de Prática**: Carrossel com os serviços jurídicos oferecidos
5. **Contato**: Seção para agendamento de consultas
6. **Estatísticas**: Números de casos bem-sucedidos e clientes satisfeitos
7. **FAQ**: Perguntas frequentes sobre serviços jurídicos
8. **Footer**: Informações de contato, localização e horário de funcionamento


## 📱 Responsividade

O template foi desenvolvido seguindo o conceito de Mobile First, garantindo uma experiência consistente em:
- Smartphones
- Tablets
- Desktops
- Telas grandes
